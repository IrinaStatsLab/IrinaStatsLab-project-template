# output/rds/

Intermediate R objects saved between pipeline steps (e.g., cleaned data frames, fitted models).

Save with `saveRDS()` and load with `readRDS()`:

```r
# Save
saveRDS(my_object, "output/rds/my_object.rds")

# Load in a downstream script
my_object <- readRDS("output/rds/my_object.rds")
```

Files here are generated by scripts and should be reproducible — safe to delete and regenerate.
